About

BASE Advanced Excel Workshop

The Business Association of Supply Expertise(BASE) is a Rutgers Business School student organization focusing on preparing students for entering the world of supply chain. This program is created and ran by the Newark's BASE organization which offeres the certification for completion of the multi-week program and passing the organization administed exam. Student's participating in the program must complete BASE's Excel Level 1 program prior to joining the program. Anyone is welcome to access materials in the guides section of this site to prepare for working with Excel as well as Level One Excel content to help develop their skills, but will not be rewarded with certification unless they've completed the program.

Overview

What to Expect

Throughout this program, students will gain skills working with data and the tools most widely used for such tasks within a business setting. The program will progressively build on prior weeks knowledge as well as final product.

Students will learn:

  • fetching data with Python,
  • cleaning data with Excel functions
  • automating the process with macros
  • generating data visualizations with PowerBI
  • performing statistical calculation in Python+Excel
These skills will be put to practical use in a final report of program's data which will automatically be generated using VBA.

Getting started

First Day Materials

RBS labs should come with required technology to complete this program. However, we ask students verify the machine they are on has the following software. Please refer to vendor website for any issues with setup or if in program follow setup steps provided on screen.

  • Microsoft Office(Excel, PowerPoint, PowerBI)
  • Anaconda Spyder
  • Rutgers Box(optional for uploading data)
